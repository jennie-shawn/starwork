# Capabilities

Capabilities are optional Core features that can be combined through presets.

They must not redefine baseline roles.

## Included in v0.1

- `starter-outputs`: `references/` + `outputs/`
- `matter-mode`: `matters/` and matter maintenance requirements
- `decisions`: high-impact decision log
- `local-identity`: project-local identity
- `local-lessons`: project-local lessons
- `main-repo-sync`: main repository to satellite workspace sync model
- `skill-mount`: soft-linked shared skills
