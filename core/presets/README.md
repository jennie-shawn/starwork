# Presets

Presets 是组装面向用户的 Core 工作区的配方。

一个 preset 会选择：

- 一个 profile
- 零个或多个 capabilities
- 默认正式事实源
- 可选的主库同步预期

Presets 主要供 CLI 和 kit 生成使用。

## v0.1 包含

- `zh-local-starter.yaml`
- `zh-local-matter.yaml`
- `zh-satellite-starter.yaml`
- `zh-satellite-matter.yaml`
- `zh-hub.yaml`
- `en-local-starter.yaml`
