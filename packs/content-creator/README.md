# Content Creator Pack

自媒体内容创作者 Pack。

这里将承载 pack 声明、skills、模板、示例和使用文档。

## v0.1 Goal

这个 Pack 不只是内容模板合集，而是一套完整内容生产工作系统。

第一版覆盖：

- 账号定位
- 选题池
- 素材库
- 爆款拆解
- 内容日历
- 单篇内容事项
- 草稿与脚本
- 发布记录
- 数据复盘
- 用户反馈
- 商业化线索
- 风格库 / 表达偏好
- 平台规则备忘

第一版闭环：

```text
灵感输入 -> 选题整理 -> 内容大纲 -> 正文/脚本 -> 发布记录 -> 数据复盘 -> 下一轮选题
```

验收标准：

- 普通内容创作者能看懂
- AI 知道素材、选题、草稿、发布、复盘分别放哪里
- 用户不需要理解底层 Core，也能直接开始工作
- 能产出一个完整 Demo 案例
